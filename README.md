## ✨ Features

- 🤖 AI-generated resume content suggestions
- 🎨 Multiple professional templates
- 👁️ Real-time preview as you type
- 📥 Export to PDF instantly
- 📱 Fully responsive design

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js, TypeScript, TailwindCSS |
| AI | OpenAI / LangChain |
| Export | PDF generation |

## 🚀 Getting Started

```bash
git clone https://github.com/aasimansari1/aasim-resume-builder.git
cd aasim-resume-builder
npm install
npm run dev
```
